## Example: NetApp Files Application Volume Group For SAP HANA with Availability Zone and Customer-Managed Keys

This example provisions an application volume group for SAP HANA using availability zone, customer-managed key encryption, and Standard network features.

### Variables

* `prefix` - (Required) The prefix used for all resources in this example.

* `location` - (Required) The Azure Region in which the resources in this example should be created.
